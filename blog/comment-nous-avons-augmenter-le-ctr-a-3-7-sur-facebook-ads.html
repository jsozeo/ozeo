Comment nous avons augmenter le CTR à 3.7% sur Facebook Ads !

client souhaitait informer son audience d’une nouvelle série d’emails quotidiens davantage segmenté. Le but étant d’augmenter le CTR (click through rate) pour augmenter le nombre de personnes qui voient la pub.

Jean-Sébastien Mancebo

CONTEXTE

Le client souhaitait informer son audience d’une nouvelle série d’emails quotidiens davantage segmenté. Le but étant d’augmenter le CTR (click through rate) pour augmenter le nombre de personnes qui voient la pub.

ACTIONS POUR AUGMENTER LE CTR ET L’AUDIENCE

Pour réaliser au mieux cette publicité, plusieurs modèles ont été testés avec plusieurs paramètres différents :

  • Différentes audiences sur Facebook Ads : similaires, retargeting, basées sur des affinités
  • Différents médias de publicités : photo, diapo, vidéo diapo avec texte, formulaire

RÉSULTATS

Dans notre cas ce qui a le mieux fonctionné a été les audiences similaires (qui s’avèrent très performante lorsque le pixel a suffisamment appris sur la cible et donc lorsque vous avez une audience relativement importante).

Ensuite ce qui a très bien fonctionné et permis d’augmenter le CTR, c’est la vidéo diapo avec du texte qui défile. Je ne sais pas pourquoi c’est ce qui fonctionne bien en ce moment, peut-être qu’il s’agit d’une mode qui passera.

COMMENT RÉUSSIR SA PUBLICITÉ DE CONVERSION

Pour réussir sa publicité et augmenter le CTR, il y a plusieurs éléments pour que ça fonctionne :

  • Mettre en place le pixel de conversion. Très important pour que le pixel puisse apprendre de l’audience qui convertis réellement.
  • Si vous avez une audience bien établie, je vous recommande fortement les audiences similaires qui fonctionnent très bien pour un certain volume d’audience. Si vous n’avez pas une audience importante, il va falloir commencer avec les audiences basées sur les intérêts.
  • Définir quel est votre objectif de conversion (une vente, un ajout au panier, la découverte d’un produit ou autre). Ce n’est pas évident qu’il faille paramétrer une vente si vous avez un ecommerce car votre pixel a besoin de data pour apprendre à reconnaître son audience, donc si vous avez une petite audience il faut commencer avec un objectif de conversion en amont de votre tunnel de vente.
  • Publier un média attractif. Une vidéo ou reprendre un post qui a bien fonctionné.
