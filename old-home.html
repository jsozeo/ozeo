Une adaptation pour la vente partenariale

Les CRM sont configurés pour la vente traditionnelle : en direct. Mais lorsqu'il s'agit de faire des vente croisées, ou via un partenaire, l'outil n'est plus adapté. Dans ce cas, nous avons créé un plugin permettant de remonter dans les dashboards d'activité commerciale, les données des ventes indirectes.
